Плагин для интеграции GigaChat AI в Obsidian. Генерируйте и улучшайте текст с помощью мощной нейросети от Сбербанка прямо в ваших заметках.

## ?? Возможности

### ?? Генерация текста
- **Обычная генерация** - создание текста с нуля по вашему промпту
- **Контекстная генерация** - работа с выделенным текстом
- **Быстрые команды** - предустановленные сценарии для частых задач

### ??? Функции работы с текстом
- **Улучшение текста** - переработка стиля при сохранении смысла
- **Перефразирование** - изложение текста другими словами
- **Объяснение** - упрощение сложных концепций
- **Суммаризация** - создание краткого содержания
- **Продолжение текста** - автоматическое дополнение начатых мыслей

### ?? Гибкие настройки
- Настройка параметров генерации (температура, длина ответа)
- Выбор модели GigaChat
- Логирование запросов для отладки
- Тестирование подключения к API

## ?? Установка

### Способ 1: Через Community plugins (рекомендуется)
1. Откройте Obsidian > Settings > Community plugins
2. Отключите "Restricted mode"
3. Нажмите "Browse" и найдите "GigaChat Text Generator"
4. Установите и активируйте плагин

### Способ 2: Через BRAT (для бета-версий)
1. Установите плагин "Obsidian42 BRAT"
2. Перейдите в Settings > Community plugins > BRAT
3. Добавьте бета-плагин по ссылке на репозиторий
4. Активируйте плагин

## ?? Настройка API

### Получение API ключа
1. Перейдите на [developers.sber.ru](https://developers.sber.ru/)
2. Зарегистрируйтесь или войдите в аккаунт
3. Создайте новое приложение в кабинете разработчика
4. Получите API ключ и client credentials

### Настройка плагина
1. Откройте Settings > GigaChat Text Generator
2. Введите ваш API ключ в поле "API Key"
3. При необходимости настройте дополнительные параметры:
   - **API URL**: `https://gigachat.devices.sberbank.ru/api/v1`
   - **Auth URL**: `https://ngw.devices.sberbank.ru:9443/api/v2/oauth`
   - **Model**: `GigaChat` (по умолчанию)
   - **Temperature**: 0.7 (рекомендуется)
   - **Max Tokens**: 1000

4. Нажмите "Тестировать подключение" для проверки настроек

## ?? Использование

### Базовые команды

#### Генерация текста с нуля
1. Нажмите `Ctrl+P` (Cmd+P на Mac)
2. Введите: `Генерация текста с GigaChat`
3. Введите промпт в открывшемся окне
4. Нажмите "Сгенерировать"

#### Генерация из выделенного текста
1. Выделите текст в заметке
2. Нажмите `Ctrl+P`
3. Введите: `Генерация из выделенного текста`
4. Выберите быструю команду или введите свой промпт
5. Нажмите "Сгенерировать"

### Быстрые команды для выделенного текста

| Команда | Описание | Пример использования |
|---------|----------|---------------------|
| **?? Улучшить** | Улучшает стиль и читаемость текста | Технический текст > Более гладкий и понятный |
| **?? Перефразировать** | Переписывает текст другими словами | Академический текст > Разговорный стиль |
| **?? Объяснить** | Объясняет сложные концепции простыми словами | Сложное определение > Простое объяснение |
| **?? Суммаризировать** | Создает краткое содержание | Длинная статья > Ключевые пункты |

### Горячие клавиши (настраиваемые)
Назначьте горячие клавиши для частых операций:
- Settings > Hotkeys > Найдите "GigaChat"
- Назначьте сочетания для:
  - `generate-text`
  - `generate-from-selection`

## ?? Настройки генерации

### Параметры API

| Настройка | Описание | Рекомендуемое значение |
|-----------|----------|------------------------|
| **API Key** | Ваш ключ GigaChat API | Обязательно для заполнения |
| **Model** | Модель GigaChat | `GigaChat` |
| **Temperature** | Креативность ответов (0.0-1.0) | 0.7 |
| **Max Tokens** | Максимальная длина ответа | 1000 |
| **API URL** | Endpoint для генерации | Стандартное значение |
| **Auth URL** | Endpoint для аутентификации | Стандартное значение |

### Параметры температуры
- **0.1-0.3**: Консервативные, предсказуемые ответы
- **0.4-0.7**: Сбалансированная креативность (рекомендуется)
- **0.8-1.0**: Высокая креативность, неожиданные ответы

## ?? Устранение неполадок

### Частые проблемы

**? "Не удалось получить access token"**
- Проверьте правильность API ключа
- Убедитесь, что аккаунт активирован на developers.sber.ru
- Проверьте настройки Client ID и Scope

**? "API error: 429"**
- Превышены лимиты запросов
- Подождите несколько минут перед повторным запросом

**? "Не удалось отправить тестовый запрос"**
- Проверьте подключение к интернету
- Убедитесь, что API endpoints доступны

### Логирование
Включите логирование в настройках для диагностики:
- Settings > GigaChat Text Generator > "Включить логирование"
- Просматривайте логи в секции "Лог подключения"
